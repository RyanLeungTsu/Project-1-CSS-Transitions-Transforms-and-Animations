What are
C S S
animations?

Animations in CSS are independent from Javascript and are used for creating animations using various properties. They allow for smooth transitions between pages, different elements and can be used to create a certain flow to enhance user experience on your site.

How to Use
Animations

Code Description
animation-name indicates the name of the animation that is set by coder
animation-duration Inidicates how long the animation should take to complete
animation-delay Indicates how long of a delay before the animation starts.
animation-iteration-count Inidcates how many times an animation runs
animation-direction Indicates direction the animation runs in
Values include: normal, reverse, alternate, alternate-reverse
animation-timing-function Indicates how fast the animation speed will fluctuate
Values include: ease, linear, ease-in, ease-out, ease-in-out
animation-fill-mode Indicates whether the animation retains styles after its execution
Values include: none, forwards, backwards, both
@keyframes Controls various points of an animation like a flipnote animation, with the start at 0% and ending with 100%
