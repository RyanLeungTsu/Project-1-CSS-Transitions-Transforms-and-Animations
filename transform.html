CSS Transform

What does Transform do in CSS?

The transform property allows you to change an element. This property allows you to scale, rotate, move, and skew elements. Transform happens when an element changes states, such as on mouse-hover or mouse-click.

Benefits of using Transform property

  • The transform property shifts or adjusts elements without affecting the flow of the document.
  • Transformations can resize an element without affecting surrounding elements, unlike adjusting width or height.
  • You can create visually interesting effects, like tilting or skewing elements, that enhance the aesthetics of your design.
  • Add 3D effects or perspective to create immersive designs.

Transform Values

Scale

The scale value allows you to increase or decrease the size of an element.

Rotate

The element rotates clockwise or counterclockwise by a specified number of degrees. A positive value, such as 90deg, rotates the element clockwise, while a negative value, such as -90deg, rotates it counterclockwise.

Translate

The translate value moves an element left/right and up/down. The movement is based on the parameters given for the X (horizontal) Y (vertical) axes.

Skew

With the skew value, the element skews (or tilts) one direction or the other based on the values given for the X and Y axes.
